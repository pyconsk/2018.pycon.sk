Naši sponzori

Ďakujeme všetkým sponzorom, bez ktorých by nebolo možné usporiadať túto úžasnú konferenciu. Prečo sa stať sponzorom?

Platinoví

 

Python Software Foundation

Python Software Foundation je nezisková organizácia, ktorá sa aktívne venuje presadzovaniu a podpore programovacieho jazyka Python. PSF podporuje a povzbudzuje Python akcie po celom svete s poslaním rozširovania medzinárodného spoločenstva programátorov jazyka Python. PSF zároveň každoročne usporiadáva severoamerickú konferenciu PyCon.
 

Zlatí

 

Kistler

Kistler Group je švajčiarska high-tech spoločnosť - globálny líder vo svete dynamických meracích technológií. Vyvíjame a dodávame senzory, elektroniku a softvér. S našími riešeniami tak pomáhame zákazníkom vyrábať produkty s vyššou kvalitou, vyvíjať autá, ktoré produkujú nižšie emisie a zvyšovať priepustnosť cestnej infraštruktúry. V Bratislave vyvíjame softvér pre budúcu generáciu IoT zariadení. Za naším úspechom stojí vášeň pre technológie, inovatívny prístup a pracovné prostredie, ktoré podporuje tímovú prácu. Vyvíjame vozidlá, ktorým ľudia môžu veriť; vytvárame meracie systémy, ktoré zachraňujú životy. Kistler pomáha vytvárať lepšiu budúcnosť.

 

Partners

 

JetBrains

Softvérové riešenia, ktoré prináša JetBrains, robia vývojárov viac produktívnymi tým, že im zjednodušujú zložité úlohy, automatizujú tie bežné a pomáhajú im osvojiť si osvedčené postupy pri vývoji. PyCharm je integrované vývojové prostredie pre profesionálnych vývojárov od JetBrains, poskytujúce sadu nástrojov pre produktívny vývoj. PyCharm je poskytovaný v dvoch verziách. Open-source PyCharm Community Edition je zdarma a vhodný na programovanie čisto v Pythone, zatiaľ čo PyCharm Professional Edition je vytvorený pre profesionálov v oblasti vývoja Web aplikácií, dátových analytikov a Python developerov.
